# 03. 연속 몰입 블록 길이 ↔ 순위

> **명제** · 순공시간 중 끊김 없는 연속 몰입 블록의 길이가 길수록 순위가 높다  
> **카테고리** A · 몰입시간 × 성과 · **상태** 🟡 데이터확인필요 · **데이터** 🟨 확인필요 · **출처** 시트2-4

## 가설
순공시간 중 끊김 없는 연속 몰입 블록의 길이가 길수록 순위가 높다.

## 필요 데이터
- `출결 raw(입퇴실·외출 로그, attendance/outing_log)`
- `rank`

**가용성**: 소스 확인 필요

## 분석 방법
일별 집계가 아닌 raw 출결에서 연속 착석 구간을 재구성 → 최장/평균 블록 길이 지표화 → 순위와 상관(총 몰입량 통제).

## ⚠️ 교란요인 · 주의
student_daily_report는 일별 합계라 블록 정보 없음 → raw 출결 스키마 확인 필요. 총량 통제가 핵심.
